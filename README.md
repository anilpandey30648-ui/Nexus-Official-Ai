# AI Chat – Sarvam AI + Groq

A simple web app that lets you chat with **Sarvam AI** (free Indian language model) and **Groq** (ultra-fast open‑source models) using a clean interface.

## Features
- **Two providers:** Sarvam AI or Groq.
- **Select a model:** from available free models.
- **Enter API keys** directly in the app (or via environment variables).
- **Chat history** persists during the session.
- **Deployable** on Streamlit Cloud, Hugging Face Spaces, or run locally.
